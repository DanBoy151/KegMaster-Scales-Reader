# KegMaster Scales Reader

Simple Python project to read KegMaster scales data.

Dependencies:
PyYAML: pip install pyyaml


How to run:

Update scale.yaml to hold your scale & keg information
install dependencies from requirements.txt: pip install -r requirements.txt

```pwsh
python src\kegmaster-scales-reader\main.py
```
